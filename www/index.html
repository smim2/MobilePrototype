How to use the prototype

This prototype is a work in progress so not every feature works as intended!

In order to view the current functionality, please click the demo button, which will simulate your location being in Worcester city campus

After navigating to Worcester, please click the "Defibrillator" button to simulate directions to the nearest defibrillator!

User Notice; this application uses Google Maps API services and it is therefor important you as a user read Googles privacy and Data collection policies before allowing the use of location services (Although not currently functioning)

Googles Privacy policy and data notice can be found here: HOW GOOGLE USES LOCATION INFORMATION
